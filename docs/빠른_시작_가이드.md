## 빠른 시작

### 에셋 사용하기

설정에서 에셋 가져오기:

```typescript
import { BRAND_ASSETS, ICON_ASSETS } from '@/assets/config'

<img src={BRAND_ASSETS.primary} alt="Brand Logo" />
<img src={ICON_ASSETS.ai} alt="AI Icon" />
```

### CSS 변수 사용하기

전역적으로 사용 가능한 모든 변수:

```css
.my-component {
  background: var(--bg-secondary);
  color: var(--text-primary);
  padding: var(--space-8);
}
```

## 디자인 토큰

### 색상 팔레트

| 변수 | 값 | 용도 |
|------|-----|------|
| `--bg-primary` | #000000 | 메인 배경 |
| `--text-primary` | #ffffff | 메인 텍스트 |
| `--primary-blue` | #3b82f6 | 강조 색상 |
| `--success` | #34d399 | 성공 상태 |
| `--danger` | #f87171 | 에러 상태 |

### 간격

| 변수 | 값 | 용도 |
|------|-----|------|
| `--space-4` | 16px | 기본 패딩 |
| `--space-6` | 24px | 큰 패딩 |
| `--space-8` | 32px | 섹션 간격 |

### 타이포그래피

| 변수 | 값 | 용도 |
|------|-----|------|
| `--text-base` | 0.85rem | 본문 텍스트 |
| `--text-lg` | 1rem | 큰 텍스트 |
