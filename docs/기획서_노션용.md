> 팀원들과 함께 채워갈 기획서입니다. 🔴빨간색 항목은 근거 자료가 필요한 부분입니다.

## 1. Executive Summary

### 한 줄 요약

> "뉴스를 보지 말고, 흐름을 설계하세요."

Narrative-AI는 투자를 시작하고 싶지만 **어디서부터 시작해야 할지 모르는 사람들**을 위한 **맥락 기반 투자 시나리오 큐레이션 플랫폼**입니다.

### 핵심 가치 제안 (Value Proposition)

| 기존 방식                   | Narrative-AI                                         |
| --------------------------- | ---------------------------------------------------- |
| 종목 추천: "이 주식 사세요" | 논리 제안: "이 시나리오에 동의하면, 이렇게 담으세요" |
| 정보 과부하 → 포기         | 큐레이션 → 가이드                                   |
| 매수 후 방치                | 전제 훼손 시 Exit 알림                               |

## 2. 문제 정의 (Problem Statement)

### 2.1 핵심 문제

> "시간을 쏟을 마음은 있는데, 어디서 시작해야 할지 모른다."

```
┌─────────────────────────────────────────────────────────┐
│   💰 목돈 형성                                          │
│        ↓                                               │
│   📈 재테크 해야 한다는 의지                           │
│        ↓                                               │
│   ❓ 어디서 시작? 뭘 공부? → [진입 장벽]              │
│        ↓                                               │
│   📚 책/유튜브 → 정보 과부하 → 포기                   │
└─────────────────────────────────────────────────────────┘

```

### 2.2 Attention Economics 관점

> 🟢 [근거 필요] 관련 통계/연구 자료

| 관점                                  | 설명                                   |
| ------------------------------------- | -------------------------------------- |
| **시간 = 가장 희소한 자원**     | 퇴근 후 투자 공부할 여력이 없는 직장인 |
| **돈이 걸려있으니 의지는 있음** | 하지만 "어디서부터?"가 막막            |
| **정보 과부하 역설**            | 정보가 많아질수록 의사결정이 어려워짐  |

[Financial Attention and Household Consumption Upgrading](https://www.mdpi.com/2227-7072/13/2/95)

🟢 **시간 = 가장 희소한 자원**

정보는 넘치지만, 가계의 인지 자원과 주의력은 제한적이며 금융 정보에 얼마나 주의를 배분하느냐가 소비·투자 행동을 결정한다

🟢 **돈이 걸려 있으니 의지는 있음**

금융 Attention이 높은 가구일수록 장기적 효용을 고려한 합리적 소비 구조로 이동하는 경향이 확인됨

🟢 **정보 과부하 역설**

다만 금융 Attention 증가는 동시에 위험 인식을 강화하여 소비·투자 결정을 지나치게 보수적으로 만들 수도 있음

→ 정보는 많을수록 ‘정리·가이드’가 필요함

### 2.3 현재 대안들의 한계

| 대안                    | 문제점                           |
| ----------------------- | -------------------------------- |
| **유튜브/리딩방** | 자극적, 신뢰도 낮음, "찍어주기"  |
| **투자 서적**     | 시간 소요 많음, 실전 적용 어려움 |
| **증권사 MTS**    | 차트/숫자 나열, 맥락 없음        |
| **뉴스/경제지**   | 파편화, 무엇이 중요한지 모름     |

## 5. 솔루션 (Our Solution)

### 5.1 핵심 기능

### 1️⃣ 시나리오 마켓 (Scenario Market)

AI가 매일 시장을 분석하여 **3~5개 핵심 투자 테마**를 카드로 제시

| Before (기존 MTS)     | After (Narrative-AI)                           |
| --------------------- | ---------------------------------------------- |
| 거래량 상위 종목 나열 | **[원인] → [결과] → [수혜 섹터]**카드        |
| "삼성전자 5% 상승"    | "엔비디아가 쏘아 올린 공: HBM 수혜주 시나리오" |

### 2️⃣ 시나리오 빌더 (Scenario Builder)

5단계 질문을 통해 **자신만의 투자 논리 완성**

```
Step 1: 거시 테마 선택 (What?)
        └── "금리 인하 수혜", "AI 반도체 성장" 등

Step 2: 상승 논리 구체화 (Why?)
        └── "왜 이 테마가 지금 중요한가?"

Step 3: 수혜주 선정 (Who?)
        └── AI가 관련 기업 논리적 연결

Step 4: ⭐ Exit Plan 설정 (When to Exit?)
        └── "이 전제가 깨지면 알림받기"

Step 5: 비중 설정 (How Much?)
        └── 포트폴리오 배분

```

### 3️⃣ 내러티브 대시보드 (My Narrative)

구독한 시나리오의 **논리 유효성 실시간 추적**

| 상태        | 의미                        |
| ----------- | --------------------------- |
| 🟢 ON_TRACK | 핵심 전제 유지 중           |
| 🟡 WARNING  | 전제 흔들림                 |
| 🔴 BROKEN   | 전제 붕괴 → 매도 검토 권장 |

### 5.2 차별화 포인트

> "AI가 정답을 맞히려 하지 않고, 선택지를 제시하고 최종 결정은 사용자에게 위임"

* ✅ Legal 리스크 최소화 (투자 권유 아님)
* ✅ 사용자 교육 효과 (금융 리터러시 향상)
* ✅ 심리적 안정감 (왜 샀는지 명확 → 패닉셀 방지)

    * ❌ “지금 매도 타이밍입니다”
  * [✔︎] 모든 콘텐츠는 **시나리오·가설·맥락 설명** 중심
  * [✔︎] “투자 판단의 책임은 사용자 본인” 문구 상시 고지

  **2️⃣ 개인 맞춤 투자자문 회피 ✔︎**
  **리스크 포인트:**
  사용자 자산 규모·소득·연령을 기반으로 *구체적 투자 비율*을 제안하면 위험

  * [✔︎] 사용자 정보 기반 **개별 종목 추천 로직 없음**
  * [✔︎] 비중 설정은 “사용자 직접 입력”
  * [✔︎] AI는 **예시·시뮬레이션·범위 제시까지만**

  **3️⃣ Exit 알림 기능의 법적 포지셔닝 ✔︎**
  Exit 알림은 **가장 민감한 기능**이지만,
  현재 설계는  **‘가격 기반’이 아니라 ‘전제 기반’이라서 안전한 편** .

  * [✔︎] 가격·수익률 트리거 ❌
  * [✔︎] **논리·전제 훼손 이벤트 기반 알림 ⭕**
    * 예: “금리 인하 지연 발언”, “정책 변경”
  * [✔︎] 알림 메시지에 **행동 지시 표현 금지**
    * ❌ “지금 파세요”
    * ⭕ “시나리오 전제가 흔들리고 있습니다”

  **4️⃣ AI 생성 콘텐츠 책임 한계 고지 ✔︎**

  * [✔︎] AI 생성 정보의 **불완전성·지연 가능성 명시**
  * [✔︎] 데이터 출처 공개 또는 “출처 요약 제공”
  * [✔︎] 실시간·정확성 보장 문구 ❌

  **5️⃣ 금융상품 광고 규제 회피 ✔︎**

  * [✔︎] 특정 증권사, ETF, 펀드 **직접 홍보 금지**
  * [✔︎] 제휴 상품 노출 시 **광고/정보 명확히 구분**
  * [✔︎] ‘수익’, ‘보장’, ‘확정’ 표현 금지

  **6️⃣ 분쟁 대응 문구 (Terms) ✔︎**
