## 2. 주요 내용 구성 (Key Content)

노션 페이지에 들어갈 핵심 내용과 구조입니다. 각 섹션별로 **채워야 할 질문(Question)**과 **필요한 근거(Evidence)**를 가이드합니다.

### 2-1. Problem Definition (문제 정의)

**"왜 지금 이 서비스가 필요한가?"**

* **핵심 키워드** : Attention Economy(관심 경제), Information Overload(정보 과부하), Decision Paralysis(결정 장애).
* **배경** :
* **시간 빈곤 (Time Poor)** : 직장인은 투자 공부를 하고 싶지만, 쏟아지는 뉴스(노이즈)를 필터링할 시간이 부족함.
* **신뢰 비용의 증가** : 유튜브, 리딩방 등 정보는 많으나 '검증된 논리'인지 판단하는 데 너무 많은 에너지 소모.
* **실행의 어려움** : 책이나 강의로 거시 경제는 알겠는데, "그래서 지금 뭘 사야 해?"(Action Item)로 연결되지 않음.

> **💡 [팀원과 채울 내용]**
>
> * 우리의 잠재 고객이 현재 정보를 얻는 채널의 문제점은 무엇인가? (예: 유튜브는 너무 자극적, 리포트는 너무 어렵다)
> * 그들이 '투자'를 위해 하루에 쓰는 시간은 얼마인가?

### 2-2. Target Audience (타겟 고객)

**"우리는 누구의 시간을 절약해주는가?"**

* **메인 타겟** : **"The Efficient Learner (효율적 학습자)"**
* **Profile** : 사회초년생 ~ 대리/과장급 (20후~30중반). 입사 후 목돈(Seed Money)이 모이기 시작함.
* **Behavior** :
  * 재테크 서적을 읽거나 베스트셀러 경제 도서에 관심이 많음.
  * '묻지마 투자'보다는 '내가 납득할 수 있는 논리'를 원함.
  * 하지만 직무가 바빠서 매일 주식창을 들여다볼 수는 없음.
* **Pain Point** : "나도 투자를 해야 하는데, 공부할 엄두가 안 난다. 누군가 **'맥락(Context)'만 딱 정리해서 떠먹여 줬으면** 좋겠다."

> **💡 [팀원과 채울 내용]**
>
> * 우리가 정의한 페르소나의 '일주일 루틴' 그려보기.
> * 그들이 가장 신뢰하는 정보 출처는 어디인가? (책, 특정 유튜버, 지인 등)

### 2-3. Solution: Logical Narrative (해결책)

**"종목(What)이 아니라 논리(Why)를 팝니다."**

* **Value Proposition** : **High Context, Low Time.** (높은 맥락, 적은 시간)
* **Core Concept** :

1. **Context Curation** : AI가 파편화된 뉴스를 모아 '하나의 완성된 시나리오(기승전결)'로 구조화.
2. **Actionable Guide** : 시나리오에 동의한다면? → 바로 실행 가능한 수혜 섹터/종목 매핑.
3. **Dynamic Tracking** : "그 논리가 여전히 유효한가?"를 지속적으로 추적하고 알림. (Reason-based Exit)

> **💡 [팀원과 채울 내용]**
>
> * 경쟁사(토스, 증권사 MTS 등) 대비 우리의 차별점 한 줄 요약.
> * 사용자가 우리 서비스를 쓰고 나서 얻게 될 '감정'은? (예: 안도감, 확신, 스마트해진 기분)

### 2-4. Service Flow (서비스 흐름)

**"사용자는 어떻게 설득되고 행동하는가?"**

1. **Awareness (탐색)** : **Scenario Market**

* 홈 화면에서 매력적인 '투자 테마 카드'를 발견. (마치 넷플릭스 콘텐츠 고르듯)
* 예: *"트럼프 당선 시, 방산주가 웃는 진짜 이유 3가지"*

1. **Engagement (학습 & 동의)** : **Scenario Builder (Interaction)**

* 단순 읽기가 아님. 5단계 질문을 통해 사용자 스스로 논리를 검증.
* "이 전제 상황(금리 인하)이 올 거라고 생각하시나요?" (User: YES)
* **효과** : 사용자가 직접 참여(Confirm)했기 때문에 투자에 대한 책임감과 확신(Conviction)이 생김.

1. **Action (투자 실행)** :

* 논리가 완성되면, 관련 종목 리스트(Basket) 제공.
* 비중 조절 및 모의/실전 투자 연결.

1. **Retention (관리)** : **Narrative Dashboard**

* 전제 조건이 훼손되면 알림 (예: "금리 인하 전망치 하향 조정됨 → 시나리오 재검토 필요").
* 사용자를 계속 서비스에 머물게 하는 락인(Lock-in) 장치.

## 3. 근거 및 데이터 (Evidence for Notion)

팀원들과 노션 페이지를 채울 때, 설득력을 높이기 위해 **반드시 확보해야 할 근거 자료** 리스트입니다.

### 3-1. 시장 검증 (Why Now?)

* **[정량] 서점 베스트셀러 순위** : 경제/경영 분야 상위권 책들이 '종목 추천'인지 '거시경제/마인드셋'인지 분석. (후자라면 사람들이 '원리/논리'를 원한다는 증거)
* **[정량] 재테크 유튜브 조회수 추이** : 깊이 있는 분석 채널(슈카월드 등) vs 단순 리딩방 채널의 성장세 비교.
* **[정성] 어텐션 이코노미 관련 아티클** : "현대인의 집중력 저하와 요약 서비스(뉴스레터 등)의 부상" 트렌드 자료 스크랩.

### 3-2. 타겟 니즈 (Why Us?)

* **FGI (인터뷰) 요약** : "재테크 책을 샀지만 끝까지 못 읽은 경험", "좋다는 주식을 샀는데 왜 올랐는지 몰라서 불안했던 경험"에 대한 목소리.
* **기존 서비스 페인포인트** : 토스/카카오페이 증권의 커뮤니티(종토방)가 얼마나 노이즈가 심하고 비논리적인지 캡처. -> **"우리는 Clean Zone이다"** 강조.

### 3-3. 솔루션 타당성 (How?)

* **유사 성공 사례** :
* **MasterClass / Class101** : '지식'을 큐레이션해서 파는 모델의 성공.
* **Newneek / LongBlack** : 뉴스를 '맥락' 있게 전달해서 팬덤을 만든 사례.
* 우리는 이것의 **"금융 버전(Financial Version)"**임을 어필.
* **기술적 실현 가능성 (Feasibility)** :
* Narrative-AI가 사용하는 LLM의 추론 능력 예시 (RAG를 통해 뉴스를 어떻게 구조화할 수 있는지 데모 스크린샷).
